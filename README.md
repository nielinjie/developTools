### DevelopTools

这个工具主要有三块部分，它们并不是密切联系的整体，而是有相对独立的用途。

* model - 简化的领域模型、它的简单dsl以及简单的模型展示工具。
* diff - 以（java）方法为单位的diff。
* server - scala unfiltered为基础的web server。model和diff有些展示逻辑可以运行在这个上面。
